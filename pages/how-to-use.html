Instruções de Uso

Bem-vindo ao pInk, seu catálogo completo de quadrinhos! Esta página contém todas as informações necessárias para aproveitar ao máximo nossa plataforma.

Navegação Básica

O pInk possui uma interface intuitiva projetada para facilitar sua experiência. A navegação principal inclui:

  • Logo pInk: Clique para retornar à página inicial
  • Barra de Pesquisa: Localizada no header para buscas rápidas. Use a barra de pesquisa para encontrar quadrinhos específicos. A pesquisa funciona em tempo real e busca pelo título dos quadrinhos.
  • Filtros: Botão de filtro para refinar sua busca
  • Alternador de Visualização: Mude entre grade e lista

Filtros Avançados

Os filtros permitem refinar sua busca de forma precisa. Atualmente suportamos filtros por:

  • Editora: Selecione uma ou mais editoras
  • Ano de Lançamento: Escolha o período desejado
  • Idioma: Filtre por idioma do quadrinho

Nota: Esta funcionalidade ainda está em desenvolvimento e pode apresentar alguns bugs.

Alternar Layout

O pInk oferece duas visualizações diferentes para melhor adequação às suas preferências:

  • Visualização em Grade:
    Ideal para explorar capas e ter uma visão geral
  • Visualização em Lista: Melhor para ler títulos e detalhes completos

Use os botões de alternância no canto superior direito para mudar entre os modos.

Visualizar Detalhes

Clique em qualquer quadrinho para ver seus detalhes completos. O modal mostra:

  • Capa em alta resolução
  • Sinopse completa
  • Metadados do quadrinho (série, gênero, ano, tamanho de download, idioma)
  • Créditos (quando disponível)

Download de Quadrinhos

Para fazer download de um quadrinho:

  • Abra os detalhes do quadrinho desejado
  • Clique no botão "Download" no modal
  • Você será redirecionado para o site ranoz.gg
  • Clique em download e espere o processamento do próprio

Observação: Você pode ser redirecionado para anúncios através do ranoz.gg, estes anúncios NÃO são de minha responsabilidade.

Solicitar Quadrinhos

Não encontrou o quadrinho que procura? Use nossa ferramenta de sugestões:

  • Localizada no rodapé da página
  • Basta digitar o nome do quadrinho desejado
  • Clicar em "Solicitar" para enviar sua sugestão

Você será redirecionado para uma página do gmail, onde poderá enviar sua sugestão. Após isso, sua sugestão será analisada e adicionada ao acervo quando possível.
